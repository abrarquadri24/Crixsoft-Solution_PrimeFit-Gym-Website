# 💪 Fitlife — Gym & Fitness Website

**"Work Hard To Get Better Life."**

A modern, fully responsive single-page gym website showcasing facilities, classes, and membership plans — built with **HTML5, CSS3, and JavaScript**.

Built as part of the **Crixsoft Solution Web Development Internship** 🚀

## 📸 Preview

![Fitlife Preview](./readme-images/desktop.png)

## ✨ Features

- 🏠 **Hero Section** — bold intro with call-to-action and animated stat badges (heart rate, calories)
- 👤 **About Us** — gym introduction with coach highlight and banner imagery
- 🏋️ **Classes Section** — scrollable card layout of available fitness classes with icons
- 📰 **Blog Section** — latest fitness articles/updates preview cards
- 📞 **Contact Section** — footer with contact details and clock widget
- 📱 **Responsive Navigation** — hamburger menu for mobile devices
- 🎨 **Smooth Scroll & Animations** — polished scrolling experience across sections

## 🛠️ Built With

- **HTML5** — semantic, accessible markup
- **CSS3** — responsive layout using flexbox/grid, custom properties
- **JavaScript (Vanilla)** — mobile nav toggle, scroll behavior, and UI interactivity
- **Ionicons** — icon set used throughout the site

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- [Git](https://git-scm.com/downloads) installed on your machine

### Run Locally

```bash
# Clone the repository
git clone https://github.com/<abrarquadri>/Crixsoft-Solution_PrimeFit-Gym-Website.git

# Navigate into the project folder
cd Crixsoft-Solution_Fitlife-Gym-Website

# Open index.html in your browser
```

No build steps or dependencies required — it's pure HTML, CSS, and JS.

## 📱 Responsive Design

The layout adapts across breakpoints — from large desktop screens down to mobile — with a collapsible navigation menu and a reflowing grid for the classes and blog sections.
